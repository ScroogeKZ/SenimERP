Статус интеграции

Связь доменов

При клике "Выиграть сделку" CRM API отправляет в BullMQ событие deal.won. ERP-система получает событие и создаёт:

  • Черновик счёта (1,500,000 ₸)
  • Черновик акта выполненных работ (для услуги)
  • Черновик накладной (для товаров)

Демонстрационный сценарий:

  1. Авторизуйтесь через SSO
  2. Нажмите "Выиграть сделку"
  3. Нажмите "Войти в ERP" вверху
  4. Подпишите ЭСФ / Акт в ERP через мок NCALayer
  5. Проведите оплату в ERP
  6. Вернитесь в CRM и проверьте статус оплаты!
